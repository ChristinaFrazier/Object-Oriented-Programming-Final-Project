Food Delivery Management System - Stage 4 (GUI Application)

Project Information:

Course: CS 2210
Stage: 4 (GUI Application with File Persistence)
Team Members: Evan Darmanto, Lauren White

How to compile:

1. Make sure all .java files are in the same directory
2. Open terminal/command prompt in that directory
3. Run: javac *.java

How to run:

After successful compilation, run: java Main

System Features:

1. Customer Management GUI:
   - Add, view, update, and delete customers
   - Manage customer allergies
   - All changes automatically saved to data/customers.txt

2. Delivery Driver Management GUI:
   - Add, view, update, and delete drivers
   - Add and view driver ratings
   - Assign drivers to orders
   - All changes automatically saved to data/drivers.txt

3. Order Management GUI:
   - Create new orders
   - View order details
   - Update order status
   - Delete orders
   - View all orders
   - All changes automatically saved to data/orders.txt

4. Payment Management GUI:
   - Create payments
   - Process and confirm payments
   - Generate detailed receipts
   - All changes automatically saved to data/payments.txt

5. Support Staff Management GUI:
   - Add and view support staff
   - Assign issues to staff
   - Update issue status
   - Resolve issues
   - View staff customer information
   - Delete support staff
   - All changes automatically saved to data/staff.txt

6. Restaurant Management GUI:
   - View restaurants and menus
   - Browse menu items by category

7. Search Functionality:
   - Search for customers, drivers, and orders
   - Filter results based on search criteria

8. Report Generation:
   - Generate comprehensive system reports
   - View statistics for all entities

9. File Persistence:
   - All data automatically saved to text files in data/ folder
   - Data loads automatically when program starts
   - Data persists between sessions
   - Files: customers.txt, drivers.txt, orders.txt, payments.txt, staff.txt, vehicles.txt
