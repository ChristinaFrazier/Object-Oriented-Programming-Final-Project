Food Delivery Management System

Project Information:

Course: CS 2210
Stage: 3 (Command Line Application)
Team Members: Evan Darmanto, Lauren White

How to compile:

1. Make sure all of the .java file are in the same directory
2. Open terminal/command prompt in that directory
3. Run javac *.java

How to run:

After successful compilation, run java Main

System Features:

1. Customer Management:
- Add, view, update, and delete customers
- Manage customer allergies

2. Delivery Driver Management:
- Add and view drivers
- Manage driver ratings

3. Restaurant Management:
- View restaurant and menus
- Filter menu items by category

4. Order Management:
- Create and track orders
- Update order status

5. Support Staff Management
- View and add support staff
- Assign and resolve customer issues

6. Payment Management:
- Create and process payments
- Generate receipts

7. System Reports
- View comprehensive statistics for all entities
